# 1. Python 简介

## 什么是 Python？

Python 是一种**高级、解释型、面向对象**的编程语言，由 Guido van Rossum 于 1991 年创建。

## Python 的诞生故事

> "Life is short, you need Python."
> — Bruce Eckel

1991 年，Guido van Rossum 在荷兰创造了 Python。名字来源于他喜爱的电视剧《Monty Python's Flying Circus》，而非Python（蛇）。

## Python 的特点

### 1. 简洁易读

Python 的设计哲学强调代码的可读性和简洁性：

```python
# Java 打印 Hello World
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

# Python 打印 Hello World
print("Hello, World!")
```

### 2. 跨平台

Python 可以运行在 Windows、macOS、Linux 等各种操作系统上。

### 3. 丰富的库

Python 拥有庞大的标准库和第三方包生态：

```
📦 pip install requests
   └── urllib3
   └── certifi
   └── charset-normalizer
```

### 4. 多范式支持

Python 支持多种编程范式：

| 范式 | 示例 |
|------|------|
| 过程式 | `print("Hello")` |
| 面向对象 | `class Dog: ...` |
| 函数式 | `map(lambda x: x*2, [1,2,3])` |

## 第一个 Python 程序

```python
# 这是我的第一个 Python 程序
name = input("请输入你的名字：")
print(f"你好，{name}！欢迎学习 Python！")
```

运行效果：

```
请输入你的名字：张三
你好，张三！欢迎学习 Python！
```
