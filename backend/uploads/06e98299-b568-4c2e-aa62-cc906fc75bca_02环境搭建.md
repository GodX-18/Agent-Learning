# 2. 环境搭建

本章将介绍如何搭建 Python 开发环境。

## 安装 Python

### 方法一：官网下载

1. 访问 [Python 官网](https://www.python.org/downloads/)
2. 下载最新版本的 Python 3.12.x
3. 运行安装包，**记得勾选 "Add Python to PATH"**

```bash
# 验证安装
python --version
# Python 3.12.0
```

### 方法二：使用 Homebrew（macOS）

```bash
# 安装
brew install python@3.12

# 验证
python3 --version
```

### 方法三：使用 pyenv（推荐）

pyenv 可以管理多个 Python 版本，非常适合开发需求。

```bash
# 1. 安装 pyenv
brew install pyenv

# 2. 添加到 shell 配置（~/.zshrc）
echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.zshrc
echo 'command -v pyenv >/dev/null || export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.zshrc
echo 'eval "$(pyenv init -)"' >> ~/.zshrc

# 3. 重新加载 shell
source ~/.zshrc

# 4. 安装 Python
pyenv install 3.12.0

# 5. 设置全局版本
pyenv global 3.12.0

# 6. 验证
python --version
```

## 安装 VS Code + Python 扩展

### 1. 安装 VS Code

```bash
# macOS 使用 Homebrew
brew install --cask visual-studio-code

# 或者下载安装包：https://code.visualstudio.com/
```

### 2. 安装 Python 扩展

1. 打开 VS Code
2. 按 `Cmd + Shift + X` 打开扩展面板
3. 搜索 "Python"
4. 安装 Microsoft 提供的 Python 扩展

### 3. 配置 Python 解释器

1. 按 `Cmd + Shift + P` 打开命令面板
2. 输入 "Python: Select Interpreter"
3. 选择你安装的 Python 版本

## 虚拟环境

虚拟环境可以隔离不同项目的依赖，避免版本冲突。

### 使用 venv（内置）

```bash
# 创建虚拟环境
python -m venv myproject_env

# 激活虚拟环境
# macOS/Linux:
source myproject_env/bin/activate

# Windows:
myproject_env\Scripts\activate

# 退出虚拟环境
deactivate
```

### 使用 conda

```bash
# 创建环境
conda create -n myproject python=3.12

# 激活环境
conda activate myproject

# 退出环境
conda deactivate
```

## pip 包管理器

pip 是 Python 的包安装工具。

```bash
# 查看 pip 版本
pip --version

# 安装包
pip install requests

# 安装指定版本
pip install requests==2.28.0

# 安装多个包
pip install numpy pandas matplotlib

# 升级包
pip install --upgrade requests

# 卸载包
pip uninstall requests

# 导出依赖
pip freeze > requirements.txt

# 从文件安装
pip install -r requirements.txt
```

## 配置 pip 镜像（国内用户）

```bash
# 临时使用
pip install requests -i https://pypi.tuna.tsinghua.edu.cn/simple

# 永久配置
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

## 验证环境

运行以下代码验证环境是否正确：

```python
# env_check.py
import sys
import pip
import platform

print(f"Python 版本: {sys.version}")
print(f"Python 路径: {sys.executable}")
print(f"操作系统: {platform.system()} {platform.release()}")
print(f"pip 版本: {pip.__version__}")

# 测试常用库
try:
    import numpy
    print(f"NumPy 版本: {numpy.__version__}")
except ImportError:
    print("NumPy 未安装")

try:
    import pandas
    print(f"Pandas 版本: {pandas.__version__}")
except ImportError:
    print("Pandas 未安装")
```

## 常见问题

### Q: 显示 "command not found: python"

**A**: 确认 Python 已添加到系统 PATH，或使用 `python3` 命令。

### Q: pip 安装失败

**A**: 尝试使用 `python -m pip install xxx` 或检查网络连接。

### Q: 权限错误

**A**: macOS/Linux 使用 `sudo pip install` 或使用虚拟环境。
